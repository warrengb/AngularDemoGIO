Scene Service

The Scene Service manages scene transitions. A Scene aggregates the image, audio and code resources needed for user activity interactions. Transitions load and unload scenes with enter, exit and run states. Typically the enter task will take many cycles to complete as normally checks all assets are ready before running. Typically the exit task will clean up resources. After the enter task completes the Scene service will launch the run task.

While in Beach Scene, a request is made to the Scene service to activate Woods Scene. The Scene service gives an exit task to the Beach Scene. After the Beach Scene exit task is completed, the Scene service loads the Woods Scene with a enter task. The same steps will occur respectfully when the Woods Scene activates Beach Scene.
