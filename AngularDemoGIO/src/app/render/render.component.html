Render Service

The Render Service manages rendering of vector, raster and 3D graphics to the display. A web page can have multiple displays. An actor can have multiple render objects connected to any display. On this page you can view 2 displays. The top display is this sites menu featuring a vector renderer that draws a pie with selectable slices. The bottom display is demonstrates a pseudo 3D racing game with 3 renderers.

Display and Render Agents

Display attaches to a page canvas. A global object that get referenced by the Actor content obect and Render service. The Render agent configures the resolution, coordinate system and graphic drawing to the back buffer, a double buffering technique. The render service will call on each renderer to draw to the back buffer, then blit the buffer to the display.

The pseudo 3D racing game is built with 3 renderers The back renderer (mountains and clouds) is a parallax 2D background at a 420 X 270 pixel resolution with a 2D cartesian coordinate system. The middle renderer (road and cars) is a pseudo 3D renderer at a 1,260 X 810 pixel resolutions using a 2.5D screen coordinate system. The front renderer is a sprite 2D renderer at a 420 X 270 pixel resolution with a 2D cartesian coordinate system.

Gameplay

Drive the car inbounds avoiding oncoming vehicle collisions to view the beautiful pseudo 3D scenery. Car can steer left, right, straight and break. The following are input directions:

  • mouse Mouse
    • Hover upper center part of car to speed up forward.
    • Hover left part of car to steer left.
    • Hover right part of car to steer right.
    • Hover lower part of car to break.
  • gamepad Gamepad
    • XBox 360 Gamepads
    • Thumb pad Up - speed up forward.
    • Thumb pad Left - steer left.
    • Thumb pad Right - steer right.
    • Thumb pad Down - break.
  • keyboard Keyboard
    • Key w - speed up forward.
    • Key a - steer left.
    • Key d - steer right.
    • Key s - break.
  • fingerprint Touch
    • Touch upper center part of car to speed up forward.
    • Touch left part of car to steer left.
    • Touch right part of car to steer right.
    • Touch lower part of car to break.
